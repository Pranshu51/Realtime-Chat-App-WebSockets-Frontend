# ⚡ Realtime Chat App – WebSockets Frontend

A modern frontend application for a real-time chat platform powered by WebSockets. This project delivers instant messaging, live communication, and a smooth user experience through persistent real-time connections.

Built to demonstrate how modern frontend applications handle real-time communication using WebSocket technology and interactive UI design. Real-time chat applications commonly use WebSockets to provide low-latency, bidirectional communication between users.

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/Pranshu51/Realtime-Chat-App-WebSockets-Frontend.git

cd Realtime-Chat-App-WebSockets-Frontend
```

---

## 2. Install Dependencies

```bash
npm install
```

---

## 3. Configure Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000

VITE_SOCKET_URL=http://localhost:5000
```

---

## 4. Start Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

# 💡 How It Works

WebSockets create a persistent connection between the frontend and backend, allowing both sides to exchange messages instantly without repeatedly making HTTP requests. This enables fast and efficient real-time communication.

### Workflow

```text
User Sends Message
          ↓
Frontend WebSocket Event
          ↓
Server Receives Message
          ↓
Broadcast to Connected Users
          ↓
Realtime UI Update
```
